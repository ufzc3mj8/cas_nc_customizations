Version 2.8.3

  • Updated the link to PowerTools on the System page.

Version 2.8.2

  • Corrected the link to the Padded School Enrollments report from the Enrollment Report which prevented the padded school enrollments from displaying.

Version 2.8.1

  • Corrected an error on the System page which prevented the link to Roles Administration from displaying.

Version 2.8.0

    home.powertools.content.footer.txt
  • Added a report to identify orphaned Attendance_Time records, only available at District office.
  • Added a wizard to clean up orhaned Attendance_Time records.
  • Changed the Orphaned Standards Grades report to reflect the school in red if the school is District Office.
  • Corrected and issue with the Overlapping School Enrollments report, where it incorrectly reflected Graduated School Enrollments, and some enrollments where the entry date matched the exit date.
  • Corrected an issue with the Duplicate Entry Codes report showing duplicate exit codes.
  • Corrected an issue with the Duplicate Entry Codes Wizard, where it failed to display the number of records to be corrected.

Version 2.7.0

  • Added the ability to filter the Padded School Enrollments report to the current year.
  • Added the ability to filter the Students with no FTE's report to the current year.
  • Added a report to identify stored grades where the GPA points or the letter grade is not expected based off the grade scale used, or the GPA points of that grade do not match the gradescale based off the letter grade.
  • Added the master report for SchoolNet preparation (PowerSchool 7.x only).
  • Added a Global Preference to enable the SchoolNet Preparation Reports (PowerSchool 7.x only, District Office only).
  • Added the SchoolNet Preparation report for locating possible duplicate students.
  • Added the SchoolNet Preparation report for locating log entries with future dates.
  • Added the SchoolNet Preparation report for locating multiple teacher accounts with the same teacher number.
  • Added the SchoolNet Preparation report for locating incidents which have a future occurance date.
  • Added the SchoolNet Preparation report for viewing all attendance setting, for all schools, all years at a glance.
  • Added the SchoolNet Preparation report for locating duplicate teacher accounts by name and SSN.
  • Corrected the Misaligned Standards link on the left hand navigation menu, which previously returned an invalid page when using Custom Page Management.

Version 2.6.3

  • Added a global preference to allow Hispanic only to qualify for meeting Federal Ethnicity and Race needs.

Version 2.6.2

  • Corrected an issue with the Incomplete Scheduling Setup which caused the report to display future kindergarteners which were correctly setup.
  • Added the link to Roles Administration to the System page, which was added in PowerSchool 7.1.1.

Version 2.6.1

  • Version 2.6.1 was never publicly released. All changes made are in the 2.6.2 release notes.

Version 2.6.0

  • Added a Miscellaneous Report to identify standard conversion scales with grades longer than 8 characters.
  • Added a Miscellaneous Report to identify grade scales with duplicate cutoff percentages.
  • Added a Miscellaneous Report to identify reports with spaces at the end of the report name.
  • Added a Miscellaneous Report to identify activities with spaces in the field name.
  • Added a Miscellaneous Report to identify course groups with spaces at the end of the name.
  • Added support for the collapsing left hand navigation menu added in PowerSchool 7.1.
  • Added global sorting and formatting to the Fee Name, Grade Level, Periods, Test Score, Tests, Terms, Standards, Race Code, Teachers, Students, and School Name fields on orphaned reports to reduce the load on the database.
  • Removed the duplicate period on the Duplicate Termbins page, when referencing the Duplicate Final Grade Setup Wizard.
  • Changed the Duplicate Attendance Report to ignore records where the school does not exist.
  • Changed the Orphaned Standards report to show the standard identifier instead of the ID when the standard exists.
  • Corrected an issue where some pages would not display reports due to how page permissions were modified for DDA.
  • Corrected the non-session attendance report to redirect to the Daily Attendance page when selecting the students name on a daily record.
  • Corrected the duplicate attendance report to redirect to the Daily Attendance page when selecting the students name on a daily record.
  • Corrected an issue where the pages using DDA permissions did not load if page permissions were not set for the DDA page.
  • Corrected an issue on the Maintenance Report which did not identify duplicate preferences with null SchoolID, UserID, or YearID's.
  • Corrected an issue with the Duplicate Preferences wizard which did not remove duplicate preferences with null SchoolID, UserID, or YearID's.

Version 2.4.2

  • Corrected an issue which prevented the Orphaned Fee Transactions report from displaying the records.
  • Corrected an issue where running reports for the current year prevented reports from displaying when running them from the Enrollment Report.

Version 2.4.1

  • Corrected an issue where the global preference would not take all users with permissions to Direct Database Access to DDA when selected.

Version 2.4.0

  • Relocated the images to the root images folder. Those with image servers should put the images on their image server.
  • Added the maintenance report for public viewing. This report had previously been hidden.
  • Introduced automated wizards to correct invalid data in the maintenance reports.
  • Corrected the Orphaned Records master report to correctly filter Student Test Scores by school.
  • Corrected the Orphaned Records master report to correctly filter Termbins by school.
  • Adjusted the DDA links to redirect the user to DDE if the user does not have access to the DDA home page by page permissions.
  • Changed the left hand navigation menu to a collapsable menu. Set default to collapsed.
  • Created a user preference to display the navigation menu collapsed or expanded by default.

Version 2.3.0

  • Added a report to identify students enrolled in a school that doesn't exist, or having grade levels not within the levels of the school as the Orphaned Students Report.
  • Added invidual links to the Orphaned Records reports on the menu.
  • Added a global preference to hide the Blank Stored Grades report.
  • Added a global preference to hide the Duplicate Stored Grades report.
  • Added a global preference to hide the Padded School Enrollments report.
  • Added a link to DDE/DDA based on global preferences on the Non-In-Session Attendance report, which will take the user to the record in DDE/DDA when selecting the Attendance ID.
  • Changed the order of the Enrollment Reports. Reports now show School Enrollments prior to Section Enrollments.
  • Changed the Orphaned Reenrollments report to include records where the grade level is not within the grade levels of the school.
  • Changed the Orphaned Reenrollments report to disregard FTE's as they are identified on the Students with no FTE's report.
  • Changed the Orphaned Test Scores report to consider entries at the District Office acceptable.
  • Changed the global preference to hide Special Program Enrollments outside school enrollments. The preference has been renamed, and will also hide the Special Program Enrollments with Invalid Grade Levels report.
  • Corrected the verbiage on how to correct overlapping Special Program enrollments.
  • Corrected the Not In-Session Attendance to reflect attendance taken on the first day of a reenrollment record to be acceptable.
  • Corrected the Section Enrollments Outside School Enrollments which had previously neglected to identify students.
  • Corrected the Rows Per Page setting on the Overlapping Special Program Enrollment page, which had previously prevented the page from displaying.

Version 2.2.3

  • Corrected the sort order on the Special Programs with Invalid Grade Levels page.
  • A change log page has been added to PowerTools, which can be accessed by clicking on the version number.

Version 2.2.2

  • Added a report to identify students with no scheduling ethnicity or race.
  • Moved the preferences links higher in the menu, under the master report pages.
  • The PowerTools link in the breadcrumbs will now take the user to the home page of PowerTools, reloading the left hand menu.
  • The "Photo Management" option in the system menu which was added in PowerSchool 6.2.1.1 has been added to the PowerTools system page.

Version 2.2.1

  • Corrected the link to the Calendar Issues report on the Miscellaneous report which prevented the Calendar Issues report from opening when using Custom Page Management.

Version 2.2

  • Added the Orphaned Records Report, which displays orphaned records in many tables.
  • Separated the User Preferences and the Global Preferences to different pages, allowing page permissions to be set on Global Preferences.
  • Added a preference the the Global PReferences which allows orphaned records reports to open links in DDE and DDA.

Version 2.1.1

  • Corrected a display issue in Internet Explorer which caused the spacing in the System menu to display double spaced.

Version 2.1

  • Added the Preferences page, which provides a user to set the default number of rows, default years to display for reports with the option, and the ability to hide the Special Program Enrollments Outside School Enrollments report.
  • Added the ability to filter the Enrollment report by current year.
  • Added the ability to filter the Miscellaneous report by current year.
  • Added the ability to filter the Calendar Issues report by current year.

Version 2.0

  • Introduced the Yahoo YUI as the interface for PowerTools reporting, providing pagination and sorting capabilities.
